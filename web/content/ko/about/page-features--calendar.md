# 달력 페이지

<div class="phone-frame">
  <img alt="My Bible Log 달력 페이지 스크린샷" src="/screenshots/ko/sc9-calendar.webp" />
</div>

달력 페이지에서는 과거 성경 읽기 기록을 캘린더 형식으로 확인할 수 있습니다.

원하는 날짜를 선택하면 해당 일자의 기록을 볼 수 있습니다. 해당 날짜에 새로운 기록을 추가할 수도 있기 때문에 간편하게 과거 기록을 수정할 수도 있습니다.

캘린더의 각 날짜 칸에는 오늘의 성경 페이지와 유사한 진행 표시줄이 있습니다. 진행 표시줄 두 가지 색상으로 채워집니다.

* **진한 파란색**: 해당 일자에 새롭게 읽은 구절 (기준 시작일 이후)
* **어두운 파란색**: 해당 일자에 읽은 총 구절 수를 나타냅니다. 읽은 구절이 모두 새롭게 읽은 구절이라면 어두운 파란색 막대는 진한 파란색 막대 뒤에 가려지기 때문에 보이지 않습니다.

진행 표시줄은 일일 구절 목표 기준으로 채워집니다. 예를 들어, 일일 구절 목표가 86절이고 해당 날짜에 새로운 구절을 43절 읽었다면 새로운 구절을 나타내는 진한 파란색이 진행 표시줄 절반까지 채워집니다.

구절을 충분히 많이 읽어 일일 구절 목표를 달성하게 되면 해당 날짜에 진한 파란색 별이 나타납니다. 새로운 구절을 읽어 일일 구절 목표를 달성한 경우에는 금색 별이 대신 나타납니다.
