# 어디든 설치해보세요

My Bible Log은 홈페이지인 동시에 \"프로그레시브 웹 앱\"(PWA)이기도 합니다. 모든 기기에서 독립된 앱으로 설치할 수 있습니다 .

<div style="margin: 1rem">
  <img alt="My Bible Log 데스크톱 앱 스크린샷" src="/screenshots/ko/sc8-install-anywhere.webp" />
</div>

예를 들어, Google Chrome을 사용하면 다음과 같은 기기에 PWA를 설치할 수 있습니다.

* Windows
* Mac OS
* Linux
* Android
* iOS

PWA를 설치하기 위해서는 먼저 브라우저에서 My Bible Log를 엽니다. Chrome에서 PWA를 설치하는 방법은 아래 링크에서 확인할 수 있습니다.

[Chrome에서 PWA 설치](https://support.google.com/chrome/answer/9658361)

## 키보드 없이 데이터 입력

My Bible Log은 모바일 기기에서 편리하게 사용할 수 있도록 설계되었습니다. 많은 사람들이 이동 중에 성경을 읽거나, 읽을 수 있는 장소 어디서든 모바일 기기를 휴대하고 다니거든요.

타이핑 없이도 성경 구절을 선택할 수 있기 때문에 스마트폰, 태블릿, 기타 터치스크린 기기에서 더욱 간편하게 My Bible Log를 사용할 수 있습니다.

## 최신 상태 유지

My Bible Log는 업데이트 측면에서 일반 홈페이지와 동일하게 작동합니다.

새로고침만 누르면 항상 최신 버전을 사용할 수 있습니다.
