# 오늘의 성경 페이지

<div class="phone-frame">
  <img alt="My Bible Log 오늘 페이지 스크린샷" src="/screenshots/ko/sc4-daily-goal.webp" />
</div>

오늘의 성경 페이지에서는 일일 성경 읽기 진행 상황을 확인할 수 있습니다. 상단에는 일일 목표 달성률을 보여주는 진행 표시줄이 있습니다. 예를 들어, 일일 목표가 86절이라면 오늘 86절을 읽었을 때 막대가 가득 채워집니다.

오늘의 읽기 기록을 이곳에서 확인하고 관리할 수도 있습니다. 진행 상황을 수정해야 할 상황인가요? 기존에 입력한 항목은 언제든지 편집하거나 삭제할 수 있습니다. 하루 동안 여러 번 나누어 읽는 경우에도 유용합니다..

진행 표시줄은 다양한 색상을 통해 현재 상태를 표시합니다:

* **진한 파란색** — 오늘 새롭게 읽은 구절
* **어두운 파란색** — 과거에 읽은 구절(기준 시작일 이후)
* **반짝임 효** — 새로운 구절을 읽어 일일 목표 달성했을 때 표시

## 읽기 추

성경 어느 부분을 읽을지 고민되시나요? 오늘의 성경 페이지에서는 개인화 된 맞춤형 읽기 추천을 제공합니다.

* 새로운 구절 발견하기
* 일일 목표 달성하기
* 성경에서 여러 부분 찾아보

추천 대상은 다음과 같이 다양하게 구성됩니다.

* 신약성경
* 시편·지혜
* 기타 구약성경

추천된 장을 읽고 나면 새로운 추천이 표시되기 때문에 항상 새롭고 흥미로운 읽기 경험을 이어갈 수 있습니다.
