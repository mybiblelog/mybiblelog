# 성경 일람 페이지

<div class="phone-frame">
  <img alt="My Bible Log 성경 책 페이지 스크린샷" src="/screenshots/ko/sc7-bible-progress.webp" />
</div>

성경 일람 페이지에서는 성경 전체의 읽기 진행 상황을 확인할 수 있습니다. 상단에는 성경에서 읽은 부분을 시각적으로 나타내는 진행 표시줄이 있습니다. 단순히 왼쪽에서 오른쪽으로 채워지는 진행 표시줄과는 달리 성경 순서에 따라 실제로 읽은 위치(앞·중간·뒤)가 그대로 반영됩니다.

종이 성경책의 옆면을 연상해보세요. 이미 읽은 성경 쪽이 파란색으로 변한다고 상상해보면 바로 이런 모습일 것입니다.

진행 표시줄 아래에는 성경에 포함된 모든 책 목록이 표시됩니다. 각 책에는 다음 항목이 포함되어 있습니다.

* **회색 별**: 해당 책을 완독하면 금색으로 변합니다.
* **개별 진행 표시줄**: 해당 책을 얼마나 읽었는지 표시합니다.

## 개별 책 페이지

<div class="phone-frame">
  <img alt="My Bible Log 개별 책 페이지 스크린샷" src="/screenshots/ko/sc6-book-chapter-progress.webp" />
</div>

원하는 책을 탭하면 상세 진행 페이지로 이동합니다. 아 기능이 준비되어 있습니다.

* 성경 일람 페이지와 동일한 상단 전체 진행 표시줄
* 해당 책의 장별 격자
* 각 장에 대한 진행 표시 및 완독 별
* 각 장에서 읽은 절 수

각 장에서 두 가지 옵션이 제공됩니다.

* **열기** : 선호하는 성경 앱에서 해당 장을 엽니다. (스마폰에서는 YouVersion, PC에서는 BibleGateway)
* **기록**: 해당 장에 대한 새로운 기록을 생성합니다. 저장하기 전 세부 항목을 조정할 수 있기 때문에 다음과 같은 경우에 유용합니다:
  * 한 장에서 일부 구절만 읽은 경우
  * 여러 장을 한 번에 기록하고 싶은 경우
