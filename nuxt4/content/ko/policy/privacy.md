# My Bible Log 개인정보 처리방침

이 개인정보 처리방침은 [https://www.mybiblelog.com](https://www.mybiblelog.com)(이하 \"사이트\") 이용 시 개인정보가 어떻게 처리되는지 설명합니다.

## 수집하는 정보

사이트를 방문하면 기기에 관한 기본 정보(웹 브라우저, IP 주소, 시간대, 쿠키 등)가 자동으로 수집됩니다. 이 정보는 방문자가 사이트를 어떻게 사용하는지 파악하고 사용 경험을 개선하는 데 사용됩니다. 이를 \"기기 정보\"라고 합니다.

기기 정보는 다음을 통해 수집됩니다:

* 쿠키 — 사이트가 정상적으로 작동하도록 돕는 작은 데이터 파일
* 로그 파일 — 기본적인 사이트 사용 현황을 추적
* 웹 비콘 — 사용자가 사이트와 어떻게 상호작용하는지 파악하는 데 사용

## 정보 활용 방식

수집된 정보는 다음과 같은 목적으로 활용됩니다:

* 사이트 기능 제공·개선
* 방문자의 사이트 사용 방식 파악
* 필요 시 소통(예: 비밀번호 재설정)

## 정보 공유

수집된 정보는 다음과 같은 특정한 상황에서만 공유됩니다:

* 사이트 사용 현황을 파악하기 위해 Google 애널리틱스와 공유
  * Google의 개인정보 처리방침은 여기에서 확인할 수 있습니다: [https://www.google.com/intl/en/policies/privacy/](https://www.google.com/intl/en/policies/privacy/)
  * Google 애널리틱스 수집을 거부하려면 이곳을 방문하세요: [https://tools.google.com/dlpage/gaoptout](https://tools.google.com/dlpage/gaoptout)
* 법적 요청에 대한 대응 등 법률에 의해 요구되는 경우

수집된 정보는 누구에게도 절대 판매되지 않습니다.

## 데이터 관리

사용자는 자신의 데이터를 온전히 관리할 수 있습니다. 언제든지 계정과 관련된 모든 데이터를 삭제할 수 있습니다:

1. \"설정\" 페이지로 이동하세요
2. \"계정 삭제\" 탭을 찾으세요
3. 안내에 따라 간단한 절차를 완료하면 계정이 영구적으로 삭제됩니다

참고: 계정 삭제는 영구적이며 되돌릴 수 없습니다.

## 추적 거부(Do Not Track)

본 사이트는 브라우저에서 추적 거부(Do Not Track) 신호가 감지되어도 데이터 수집 방식을 변경하지 않습니다.

## 개인정보 처리방침 변경

본 개인정보 처리방침은 사이트 운영 방식의 변경이나 법적 요구 사항을 반영하기 위해 수시로 업데이트될 수 있습니다.
