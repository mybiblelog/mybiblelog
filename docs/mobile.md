# Mobile App

The mobile app is an [Expo](https://expo.dev) / React Native project under [`mobile/`](../mobile). Its documentation is co-located with the app code:

- **[`mobile/README.md`](../mobile/README.md)** — the primary mobile guide: development phases (Android emulator → physical device → preview APK → production), environment config, Google Sign-In, debugging, and a scripts reference. Covers **setup, local development, dev build, and publishing**.
- **[`mobile/docs/android-dev-build.md`](../mobile/docs/android-dev-build.md)** — EAS dev build for Android: EAS setup, project configuration, OAuth client IDs + SHA-1 signing.
- **[`mobile/docs/play-store-checklist.md`](../mobile/docs/play-store-checklist.md)** — Google Play Store release checklist (legal/policy, build & signing, store listing assets).
- **[`mobile/docs/offline-storage.md`](../mobile/docs/offline-storage.md)** — offline storage model and cross-version migrations.

For how mobile Google Sign-In relates to the API's OAuth configuration, see the [OAuth guide](oauth.md).
